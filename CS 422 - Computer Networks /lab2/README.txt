CS422 Lab 2

Files:
- prog_rdt.c: Main implementation of RDT 3.0 protocol.
- prog_rdt.h: Header file with structs and prototypes.
- Makefile: Builds the executable.

Compilation:
Run 'make' to compile prog_rdt.c into prog_rdt executable.
Uses gcc with -Wall -Wextra -g flags. No extra libraries needed beyond standard C (stdio, stdlib, string.h, limits.h).

Execution:
Run './prog_rdt' and input parameters as prompted (nsimmax, lossprob, corruptprob, lambda, TRACE).
